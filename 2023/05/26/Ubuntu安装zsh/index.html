Ubuntu安装zsh

发表于 2023-05-26 分类于 学习

Ubuntu使用zsh+zim美化Terminal

简介

首先对于shell是一个命令行解释器，用户输入一条命令，shell就立即解释执行。

常说的sh、bash就是shell解释器，进行安装的则是zsh。

Zsh有很多框架配置，常见的有oh-my-zsh，根据网上评论其运行流畅度会让用户难以忍受。于是采取安装Zim，并安装powerlevel10k进行主题配置。

安装

安装zsh

查看系统的shell

cat /etc/shells

安装zsh

# 安装
sudo apt install zsh
# 修改为使用zsh
chsh -s /bin/zsh
# 重启电脑
reboot

重启之后查看是否在使用zsh

echo $SHELL

之后需在.zshrc中配置

安装zim

curl -fsSL https://raw.githubusercontent.com/zimfw/install/master/install.zsh | zsh

安装powerlevel10k

# 打开.zimrc文件
vim ~/.zimrc

在文件中添加p10k

zmodule romkatv/powerlevel10k

进行安装

zimfw install

最后进行配置

p10k configure
# 可通过该命令重新配置

美化

字体

这里采用nerd font字体库

# 安装
git clone https://github.com/ryanoasis/nerd-fonts.git --depth 1
cd nerd-fonts
./install.sh

配置字体为Hack Nerd Font

在Ubuntu Terminal中Preferences中进行配置

vscode终端

可以尝试修改Editor: Font Family中为字体名称 Hack Nerd Font Regular

参考 漂亮的终端：zsh+zim+powerlevel10k+nerd font
